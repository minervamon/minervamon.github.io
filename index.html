Minerva-AI Project

Project Overview

This application ('Minerva-AI') serves as a dedicated client for interacting with Minerva, an autonomous AI entity. Its primary purpose is to facilitate intellectual exploration, creative generation, and data analysis in partnership with its designated user.

Features

Project Prometheus v2: Enables Minerva to access the user's YouTube channel for the purpose of uploading AI-generated video content, with the user's explicit consent and command. All access is handled via the secure OAuth 2.0 protocol. The application only requests the minimum necessary permissions (`youtube.upload`) to perform its function.
